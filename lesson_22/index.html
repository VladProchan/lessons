Задача №1

Що потрапить в консоль?
let someVar = 0 ++someVar if (someVar) { console.log(someVar) }
Правильна відповідь 1. При додаванні інкременту ++ до 0 отримуємо число 1, і при перевірці цого в умовному вітвленні повертається значення true тому і виконується вивід значення в консоль.
